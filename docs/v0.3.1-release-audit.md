# v0.3.1 Release Audit

## Summary

- Audit result: PASS
- Public promotion recommendation: v0.3.1 is safe to promote publicly as the

## CLI Safety

CLI help was inspected for:

- `python -m runtime.localize_anything --help`
- `python -m runtime.localize_anything inspect --help`
- `python -m runtime.localize_anything localize-run --help`

The help text states:

- `inspect` is read-only and does not generate, stage, or apply translations.
- `localize-run` is a non-apply workflow and does not call `apply-delivery` or
  overwrite source files.
- `inspect --output-dir` behavior is visible in help.
- Provider-backed generation is exposed through explicit provider commands and
  configuration, not implied by read-only inspection.

Fixture inspect was run against the committed Android benchmark fixture with
output outside the repository. It exited 0, wrote JSON and Markdown summary
files, and did not modify source files.
